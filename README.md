# Lokal AI Videostabiliserare (Streamlit)

## Installation

```bash
pip install -r requirements.txt
```

## Starta appen

```bash
streamlit run app.py
```

## Funktion

- Ladda upp `.mp4` eller `.mov`
- Stabiliserar automatiskt med VidStab
- Exporterar som `.mp4` (H.264 codec)
- Max filstorlek: 500 MB
